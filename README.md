Personal config files for 
- [Aerospace](https://nikitabobko.github.io/AeroSpace/guide#homebrew-installation)
	- i3 like tiling manager for macOS. Super useful for people that go split screen in macOS
- [Obsidian](https://obsidian.md/)
	- Note taking app with several useful plugins
- [VScode](https://code.visualstudio.com/)
	- code editor with configurations to be as minimal as possible and have vim-like keyboard shortcuts
- [Zsh (Oh My Zsh)](https://ohmyz.sh/)
	- Terminal setup with aesthetic options set up in [powerlevel10k](https://github.com/romkatv/powerlevel10k)
	- [Ghostty](https://ghostty.org/docs) is my current preferred terminal emulator
